# MDL/DMZ 종합상황실 3D 대시보드 - 구현 완료

MDL(군사분계선)과 DMZ(비무장지대) 긴장 상황을 실시간 모니터링하기 위한 군사 종합상황실 스타일의 React 대시보드 애플리케이션 구현이 모두 완료되었습니다.

---

## 1. 구현된 핵심 기능 요약

### 🌐 3D 한반도 지구본 (`Globe3D.jsx`)
- `react-globe.gl`을 활용한 3D 가상 지구본.
- 카메라를 한반도 좌표(위도 37.8°, 경도 127.0°)로 자동 집중 포커싱.
- MDL 라인(빨간색 아크), DMZ 경계 영역(반투명 펄스 링) 및 주요 GP 위치를 3D 마커 및 펄싱 포인트로 시각화.
- 자동 회전 및 줌, 인터랙티브 3D 회전 제어 기능 내장.

### 🗺️ DMZ 전역 지도 (`DMZMap.jsx`)
- `maplibre-gl`을 사용한 인터랙티브 전술 2.5D/3D 지도.
- 오픈소스 CartoDB Dark raster 타일을 적용하여 API 키 없이도 매끄러운 다크 모드 렌더링.
- 군사분계선(MDL, 빨간색 이중선), 남방한계선(SLL, 파란 점선), 북방한계선(NLL, 황색 점선) 오버레이 표시.
- 판문점, 도라전망대, 을지전망대 등 주요 관측 및 대치 거점 마커 표시 및 정보 팝업 제공.
- 창 크기 변경에 유연하게 대처할 수 있도록 **ResizeObserver** 기반 리사이징 로직 적용.

### 📰 실시간 뉴스 수집 레이어 (`NewsFeed.jsx` & `useNews.js`)
- GNews API를 이용해 한반도 안보 키워드("MDL", "DMZ", "북한", "군사")에 부합하는 뉴스를 한국어(`lang=ko`, `country=kr`)로 자동 수집.
- API 키가 설정되지 않은 경우, 즉시 현실적인 상황 연출을 위한 **모의 긴급 뉴스 데이터(Mock Data)**로 즉각 대체 작동.
- 하단 뉴스 티커 영역은 끊김 없이 흐르는 부드러운 자동 스크롤 애니메이션(마우스 오버 시 일시정지) 적용.
- 우측 사이드바에서도 주요 뉴스를 리스트로 제공.

### 🛡️ 상황 통제 패널군
- **위협 수준 게이지 (`ThreatLevel.jsx`)**: SVG 원형 링 게이지로 현재 DEFCON/위협 단계를 시각화하고 긴박한 상황에 맞는 펄스 글로우 이펙트 적용.
- **실시간 시계 (`TimeDisplay.jsx`)**: 한국 표준시(KST) 24시간 밀리터리 시간 포맷 및 협정 세계시(UTC) 동시 표시.
- **상황 요약 (`StatusPanel.jsx`)**: 경계태세(진돗개 단계 등), 감시정찰, GOP 경계 등의 상태 인디케이터 제공.
- **주요 지표 (`StatsGrid.jsx`)**: easeOutExpo 알고리즘을 이용한 부드러운 카운트업 수치 애니메이션 제공.
- **DMZ 기상 (`WeatherPanel.jsx`)**: 전방 기상 상황(체감 온도, 습도, 풍속, 시정 거리 등) 시뮬레이션 데이터 자동 생성 및 갱신.

## 2. 디자인 및 테마 시스템 (`index.css`)
- **다크 밀리터리 미학**: 깊은 네이비 블랙 배경(`#060a13`), 그린 글로우 테두리, 전술 지도의 밝은 녹색/적색/황색 컬러 조화.
- **CRT 모니터 스캔라인 효과**: 전술 상황실 CRT 화면을 연상케 하는 반복 그라디언트 스캔라인 오버레이 레이어 구현.
- **프리미엄 폰트**: 타이틀용 `Orbitron`, 수치용 `JetBrains Mono`, 텍스트용 `Noto Sans KR` 등 Google Fonts 적용.
- **반응형 그리드 레이아웃**: 좌측 상황 지표, 중앙 3D 화면(지구본 + 지도), 우측 전술 상황 정보, 하단 뉴스 피드로 구성된 3열 그리드 적용.

## 3. 실행 및 검증 방법

### 의존성 설치 및 로컬 서버 구동
1. 터미널에서 프로젝트 폴더로 이동합니다.
   ```bash
   cd c:\Users\user\MDA
   ```
2. 로컬 개발 서버를 구동합니다.
   ```bash
   npm run dev
   ```
3. 브라우저에서 제공되는 로컬 주소(예: `http://localhost:5173`)로 접속하여 화면을 확인합니다.

### 배포용 빌드 테스트
프로젝트가 정상적으로 최적화 빌드되는지 아래 명령을 통해 확인해 두었습니다.
```bash
npm run build
```
*(성공적으로 최적화된 하나의 JS/CSS 청크로 번들링되는 것을 검증했습니다.)*
